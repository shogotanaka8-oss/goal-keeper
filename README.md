# Goal Keeper 🎯

**Goal Keeper** は、IT初心者でも直感的に使える、シンプルかつ戦略的な目標管理アプリケーションです。
半期ごとの目標だけでなく、「どのフィールドで」「誰に」「どうやって」評価されるかという戦略まで深く管理できます。

## ✨ 特徴

- **期間別の目標管理**: 2030年までの上期・下期に対応。未来のキャリアプランニングにも最適です。
- **戦略的分析フレームワーク**:
  - 🗺️ **フィールド分析**: 自分が活躍すべき場所を定義
  - 👁️ **審査員 (キーマン)**: 誰に評価されるべきかを特定
  - 🏆 **勝利条件**: 評価されるための具体的な成果を定義
- **戦術の策定**:
  - 💡 **新しいアプローチ**: 独自の創意工夫やツール活用
  - 📢 **アピール戦略**: 成果を効果的に見せる方法
- **データ保存**: 入力したデータはブラウザに自動保存されます（会員登録不要）。
- **モバイル対応**: スマホからもPCからも美しく表示されます。

## 🚀 使い方

1. **目標を立てる**: タイトルと期間、詳細を入力します。
2. **戦略を練る**: フィールド分析や、評価者（審査員）を設定します。
3. **戦術を決める**: どうやって達成するか（創意工夫）、どうアピールするかを入力します。
4. **追加する**: 「目標を追加する」ボタンを押すと、カード形式でリストに追加されます。
5. **振り返る**: 定期的にリストを見返し、進捗を確認しましょう。不要になった目標は削除できます。

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: 変数を使用したモダンなデザイン、レスポンシブ対応
- **JavaScript**: LocalStorageを使用したデータ永続化
