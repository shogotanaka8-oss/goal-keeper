# 超かんたん！Git（ギット）の使い方講座

Gitをインストールされたとのことで、素晴らしいです！
これからはドラッグ＆ドロップではなく、コマンドでかっこよく更新できるようになります。

## 🔄 基本の3ステップ

Gitの作業は、基本的にこの**3つのコマンド**の繰り返しです。

### 1. 追加する (add)
変更したファイルを「次はこれを保存するぞ」と指定します。
```bash
git add .
```
（`.` は「全部」という意味です）

### 2. 記録する (commit)
メッセージ付きで変更を保存します。
```bash
git commit -m "変更内容のメモ"
```
（例: `git commit -m "色を赤に変更"`）

### 3. 送信する (push)
GitHub（インターネット上）にアップロードします。
```bash
git push
```

---

## 🛠️ 最初の設定（まだやってなければ）

Gitを入れた直後は、初期設定が必要です。ターミナルで1回だけ実行してください。

```bash
git config --global user.name "あなたのGitHub名"
git config --global user.email "GitHubに登録したメールアドレス"
```

## ⚠️ もしエラーが出たら？
VS Codeのターミナルで `git` が使えない場合、PCを**再起動**すると動くようになることが多いです（パスを通すため）。
