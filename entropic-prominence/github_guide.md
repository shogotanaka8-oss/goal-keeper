# GitHub 公開手順 (コマンド不要・ブラウザ版)

パソコンにGitソフトが入っていなくてもできる、ブラウザだけを使った一番簡単な方法です。

## 1. アップロード画面を開く
1. さきほど作ったGitHubのリポジトリページ（`https://github.com/shogotanaka8-oss/goal-keeper`）を開きます。
2. 画面の中央にあるリンク **「uploading an existing file」** をクリックします。
   （または画面上の「Add file」ボタン → 「Upload files」を選択）

## 2. ファイルをアップロードする
1. 「Drag files here...」という点線のエリアが表示されます。
2. あなたのPCのフォルダから、以下の3つのファイルをドラッグ＆ドロップで入れます。
   - `index.html`
   - `style.css`
   - `script.js`
3. ファイルが一覧に表示されたら、下の方にある緑色のボタン **「Commit changes」** をクリックします。
   （タイトルなどの入力欄は空欄のままでOKです）

## 3. 公開設定をする
1. ファイル一覧の画面に戻ったら、上部のタブメニューから **「Settings」**（歯車アイコン）をクリックします。
2. 左側のサイドメニューから **「Pages」** を探してクリックします。
3. **Build and deployment** という項目の下にある **Branch** の設定を見ます。
4. ドロップダウンメニューで「`None`」になっているところを **「`main`」** に変更します。
5. 変更したら、すぐ隣に出る **「Save」** ボタンをクリックします。

## 4. 完成！
1. 1〜2分ほど待ちます（この間に裏側で準備が進んでいます）。
2. Webページをリロード（再読み込み）します。
3. 画面の上の方に **「Your site is live at...」** というメッセージと共にURLが表示されます。
4. そのURLをクリックすれば、世界中どこからでも見れるあなたの目標管理アプリの完成です！
