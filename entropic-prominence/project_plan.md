# 半期目標管理アプリ 開発計画

IT初心者の方でも分かりやすいシンプルな構成で作成します。

## 1. アプリの概要
- **名前**: Goal Keeper (ゴールキーパー) - 仮称
- **目的**: 半期ごとの目標を設定し、振り返りを管理する。

## 2. 機能要件
1. **目標の追加**: 目標のタイトル、内容、対象期間（例：2024年上期）を入力して保存できる。
2. **一覧表示**: 登録した目標をカード形式で見やすく表示する。
3. **ステータス管理**: 「未着手」「進行中」「達成」「未達成」などの状態を変更できる。
4. **データの保存**: ブラウザにデータを保存し、画面を閉じても消えないようにする（ローカルストレージ使用）。

## 3. 開発ステップ
1. **準備**: 必要なファイル（HTML, CSS, JavaScript）を用意する。
2. **骨組み (HTML)**: 画面の要素（入力欄、ボタン、リスト）を配置する。
3. **デザイン (CSS)**: 色や配置を整え、見やすく、やる気が出るようなデザインにする。
4. **動き (JavaScript)**: ボタンを押した時の保存処理や、リストの表示処理を作る。
5. **テスト**: 実際に動かして確認する。

## 4. 技術スタック
- **HTML5**: ページの構造
- **CSS3**: デザイン（モダンで見やすいスタイル）
- **JavaScript (Vanilla)**: アプリの動作ロジック
